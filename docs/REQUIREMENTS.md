## 미션 요구사항

- [x] 숫자를 증감할 수 있는 카운터를 구현한다.
  - [x] 가능한 숫자는 0 이상, 3 이하이다.
- [ ] ? 에 hover하면 tooltip을 띄워준다.
- [ ] 스크린 리더가 정상적으로 동작하도록 한다.
  ```
    성인 탑승자 한명 줄이기 버튼
    성인 1 텍스트 숫자만 수정
    성인 탑승자 한명 늘리기 버튼
    성인 승객 추가 2
    성인 탑승자 한명 늘리기 버튼
    성인 승객 추가 3
  ```
